## Running Locally

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Building for Production

```bash
npm run build
```

Output is generated in the `dist/` folder.

## Deployment

Deployed on [Vercel](https://vercel.com). Pushing to `main` triggers an automatic redeploy once the project is connected.
